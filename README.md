# Examen Final - Proyecto

Este proyecto contiene una app mínima en HTML que muestra mi nombre, DNI y el mensaje “Examen Final”.

## Cómo ejecutarlo
Abrir el archivo `index.html` en un navegador.

## Docker

### Comandos utilizados
- docker build -t final-lautaro .
- docker run -d -p 8080:80 final-lautaro

El contenedor expone la app en http://localhost:8080
